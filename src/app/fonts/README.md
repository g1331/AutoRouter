# 内置字体

均取自 [Fontsource](https://fontsource.org/)（`@fontsource-variable/*` 包的 latin 可变字重产物），
以 SIL Open Font License 1.1 分发，可随项目自由再分发。

| 文件                                     | 字体           | 角色                                                                  | 许可    |
| ---------------------------------------- | -------------- | --------------------------------------------------------------------- | ------- |
| `manrope-latin-wght-normal.woff2`        | Manrope        | 正文 sans（`--font-sans`）                                            | OFL 1.1 |
| `jetbrains-mono-latin-wght-normal.woff2` | JetBrains Mono | 数据/代码 mono（`--font-mono`）                                       | OFL 1.1 |
| `saira-latin-wght-normal.woff2`          | Saira          | 标题/大数值 display（`--font-display`，DIN 系，fallback Bahnschrift） | OFL 1.1 |

许可全文见各字体上游仓库的 `OFL.txt`（如 <https://github.com/Omnibus-Type/Saira>）。
